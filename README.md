Цель проекта:
Программировать с удовольствием: Вместо того чтобы делать скучные "котики" и "туду-листы", я решил сразу взяться за тему, которая меня реально интересует индустрия видеоигр. Это помогает мне не бросать код на полпути.
Создать удобный каталог впечатлений: Я заметил, что часто хочу быстро найти и сравнить свои впечатления от прохождения игр («Ведьмак 3», «Elden Ring», «God of War»). На AGeeks я собрал карточки обзоров с оценками, чтобы всегда иметь под рукой свою "библиотеку восторгов".
Попрактиковаться в современных технологиях: Для меня этот проект — настоящий полигон. Здесь я применяю CSS-переменные для создания тёмной и светлой темы, использую JavaScript для динамической подгрузки контента и впервые разбираюсь с промисами и обработкой событий.
Сделать стартовую площадку для новых идей: В будущем я хочу добавить на портал не просто статичные карточки, а подключить настоящее API с играми, научиться добавлять пользовательские комментарии и сделать личный кабинет. AGeeks — это фундамент для моих будущих крутых проектов.
Проще говоря, AGeeks — это место, где мой интерес к играм превращается в реальный код и рабочий продукт. Я учусь, делюсь впечатлениями и строю что-то своё. Добро пожаловать в мою гик-лабораторию! 😎🎮
